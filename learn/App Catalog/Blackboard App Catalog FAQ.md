<h1><a id="user-content-blackboard-app-catalog---faq" class="anchor" aria-hidden="true" href="#blackboard-app-catalog---faq"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Blackboard App Catalog - FAQ</h1>

<h2><a id="user-content-1-what-is-the-blackboard-app-catalog" class="anchor" aria-hidden="true" href="#1-what-is-the-blackboard-app-catalog"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>1. What is the Blackboard App Catalog?</h2>
<p>Partner integrations continue to play a critical and vital role within the Blackboard Learn ecosystem, and we’ve always strived to maintain a current and detailed inventory of our partners and their available integrations. The new App Catalog directly enables our partner community to manage their own App listings—ultimately providing our clients with a current, comprehensive, and detailed list of available integrations across all flavors of Blackboard Learn. As our partner community continues to expand, and our clients navigate their transitions from the Original to Ultra experience of Blackboard Learn, partner integrations are more important than ever before. Our partners and clients deserved a scalable, dynamic, and comprehensive resource for partner integrations – and we’ve delivered.</p>
<h2><a id="user-content-2-how-does-the-blackboard-app-catalog-work" class="anchor" aria-hidden="true" href="#2-how-does-the-blackboard-app-catalog-work"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>2. How does the Blackboard App Catalog Work?</h2>
<ul>
<li>All Partners in the <a href="https://www.blackboard.com/partnerships/become-a-partner" rel="nofollow">Blackboard Partnerships Program</a> are listed on the <a href="https://appcatalog.blackboard.com/list-partners/" rel="nofollow">App Catalog Partners</a> page and are able to submit detailed information for each of their "Apps" (integrations) to the App Catalog. Submitted Apps are reviewed by the Blackboard App Catalog team and appear in the App Catalog upon approval.</li>
<li>The App Catalog enables anyone to browse, search, or filter the Apps by integration type (LTI + REST, Building Block, Partner Cloud) or Category (Accessibility, Admin/Support Tool, Assessment, Classroom Tool, Consulting/Dev Services, Content, Data Insights, Multimedia, Safety/Security, SIS/System Integrator).</li>
<li>The new Blackboard App Catalog uses <a href="https://openchannel.io/" rel="nofollow">OpenChannel</a> - an experienced provider of LMS integration marketplace solutions in the educational space.</li>
</ul>
<h2><a id="user-content-3-what-does-ultra-ready-mean" class="anchor" aria-hidden="true" href="#3-what-does-ultra-ready-mean"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>3. What does "ULTRA READY" mean?</h2>
<p>Any integration listed as "Ultra Ready" is labeled as such to indicate it was built leveraging only Blackboard integration frameworks specifically supported in Ultra (i.e. Partner Cloud, LTI 1.1+ and/or Learn REST APIs).  The term "Ultra Ready" is not equivalent to a Blackboard certification status for the Ultra experience.<a href="#ultraready">††</a></p>
<p><a target="_blank" rel="noopener noreferrer" href="/blackboard/blackboard.github.io/blob/Backup/learn/App%20Catalog/ultraready.png"><img src="/blackboard/blackboard.github.io/raw/Backup/learn/App%20Catalog/ultraready.png" alt="Ultra Ready Badges" title="Ultra Ready Badges" style="max-width:100%;"></a></p>
<h2><a id="user-content-4-whats-the-difference-between-the-3-app-types-lti--rest-building-blocks--partner-cloud" class="anchor" aria-hidden="true" href="#4-whats-the-difference-between-the-3-app-types-lti--rest-building-blocks--partner-cloud"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>4. What's the difference between the 3 App types (LTI + REST, Building Blocks, &amp; Partner Cloud)?</h2>
<ul>
<li><em><strong>LTI + REST:</strong></em> LTI + REST apps are those integrations that rely on and utilize <a href="https://help.blackboard.com/Learn/Administrator/SaaS/Integrations/Learning_Tools_Interoperability" rel="nofollow">LTI Tool Placements</a> and/or <a href="https://help.blackboard.com/Learn/Administrator/SaaS/Integrations/Compare_Building_Blocks_and_Rest#why-rest-apis_OTP-1" rel="nofollow">Blackboard Learn REST APIs</a>. LTI + REST apps are compatible across all flavors and delivery methods of Blackboard Learn (Original and Ultra experiences) and have no association with legacy Blackboard Building Blocks.</li>
<li><em><strong>Building Block:</strong></em> While only fully functional in the Original Experience of Blackboard Learn, for any <a href="https://help.blackboard.com/Learn/Administrator/SaaS/Building_Blocks" rel="nofollow">Building Blocks</a> that are installed on a Blackboard Learn environment with the Ultra Experience enabled, the functionality of the integration will only be available in courses using the Original course view. Building Block functionality is not compatible with the Ultra experience of Learn including anywhere within the Ultra base navigation (outside of the course context) or within Ultra view courses.</li>
<li><em><strong>Partner Cloud:</strong></em> Natively available in all flavors and delivery methods of Blackboard Learn (Original and Ultra experiences), <a href="https://help.blackboard.com/Learn/Administrator/SaaS/Integrations/Content_Market#partner-content-in-the-content-market_OTP-0" rel="nofollow">Partner Cloud</a> Apps and have no association with a legacy Blackboard Building Block. The Blackboard Partner Cloud streamlines our inventory of content integrations into a single registration and activation process, eliminates Building Block version control issues, and accelerates the delivery of new partner features and product updates. Via the course-embedded <a href="https://help.blackboard.com/Learn/Instructor/Course_Content/Create_Content/Add_Content_From_External_Sources#content-market_OTP-0" rel="nofollow">Content Market tool</a>, instructors and course builders can access ready-made material from textbook publishers to enhance their curriculum and provide students with supplementary materials that aid in learning. At the same time, sensitive student data remains private and secure.</li>
</ul>

Why can't I find the App(s) I'm looking for?</h2>

If you know of an App that you'd like to see in the App Catalog, please fill out the <a href="https://go.blackboard.com/suggestanintegration" rel="nofollow">Suggest an Integration</a> form and we will work with the partner/vendor to get the App listed.</p>
