Interface IRouteEvent

The event to send when a user navigates to another page within Ultra.

An example event when navigating to base.courses.peek.course.outline:

example

 {
     eventType: 'route',
     routeData: {
         courseId: '_555_1'
     },
     routeName: 'base.courses.peek.course.outline',
     type: 'event:event',
 }
