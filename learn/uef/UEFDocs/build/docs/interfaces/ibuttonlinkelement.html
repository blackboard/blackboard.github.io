Interface IButtonLinkElement

Element used to render a link styled as a button.

An example of a ButtonLink:

example

{
   tag: 'ButtonLink',
   props: {
     to: 'base.integration',
     className: 'uef--button'
   },
   children: 'Click me!'
}
